🚩笑脸公式【隔年增长率】

三个时间 + 两个增长率， 求隔年增长率

🚩⚠️事项

  1. 乘积部分省略：【 |X%| |Y%| 均小于 10%】
  2. 增长率正负均可以使用
  3. 时间间隔–>隔月也行，三个时间点两个增长率

正负问题，遇到补充

🚩合成增长率【两部分合成整体】

整体增长率一定在于两者之间。且偏向于基数【现期量】较大的一方

如果问 1-3 月的增长率，估算，两者之间。

高频易错

  • 增长率相关

笑脸公式

高频易错题精选001–公考齐麟

莫名其妙（时间点很奇怪）的增长率，可以把已知条件中的时间点和问题中的时间点从前到后写成一行，再将已知的数据列出来，分析问题。

笑脸公式：x%+y%+x%×y%

注意：

①x%×y%一般选择将其中一个增长率化为1/N，避免出错；

②x%、y%均小于10%时，乘积直接忽略；

③x%、y%的取值正负均可，增长率为负数时，直接将负数套入公式即可。

  • 特殊词汇
  • 比大小

遇到精确计算问题，方法有一下几个：

  1. 往特殊百分号靠近，不足的补充；用多位数x分数 + 多位数x百分数
    1. 借位，多位数x百分数，2315 * 3.2% => 23 * 3.2
  2. 提取公因数，一般拆分百分数
    1. ≥四位数 * 百分数，差0.1；就会误差很大；
    2. 提取公因数，大的百分数 * 小的整数 + 大的整数 * 小的百分数，误差都不会很大。
    3. 速算，心算继续。
  3. 整数*整数
    1. 可以化为特殊分数，
    2. 只是移动小数点

特殊词

XX率

  1. 除增长率外，都朝比重方向思考，转换为A/B=r
  2. 找注解，注释就可能是公式
  3. 关键字查找经验，找关键字，在关键字附近找数据，
  4. 步骤
    1. 先明白考试的是啥，列出来公式。确定是基期、现期
    2. 找关键字，定位数据；
    3. 百化分，截位，一直关注选项，能确定就别算。
    4. 形成习惯！！

高频易错题精选002–公考齐麟

【例 2】2012 年末，中国大陆总人口 135404 万人，全年出生人口 1635 万人，死亡人口 966 万人。 从性别结构看，男性人口 69395 万人，同比增加 327 万人，女性人口 66009 万人， 同比增加 342 万人； 从城乡结构看，城镇人口 71182 万人，同比增加 2103 万人；乡村人口 64222 万人，同比减少 1434 万人。 2012 年中国男性人口、女性人口、城镇人口的同比增长率按大小顺序正确的是： A．城镇人口＞女性人口＞男性人口 B．城镇人口＞男性人口＞女性人口 C．女性人口＞男性人口＞城镇人口 D．女性人口＞城镇人口＞男性人口

比较大小，如果能不计算精确值，就不要计算；

327/69xxx ; 342/65xxx; 2103/50xxx

前两者，分子视为相同，三个看分母视为相同。

求基期量，前两位就行了

  • 猜题一

两期比重差

  1. a > b ⬆️
  2. a < b ⬇️

类型一

今年比去年，相邻年份、月份等；

判断上升下降，后，直接猜小的

类型二

今年🆚前年，中间隔年，需要用到笑脸公式的。

  1. 第一步：快速计算识别，a 🆚 b 关系，是上升还是下降。记住那个数字。
  2. 排除那个数字的两项
  3. 排除下降/⬆️的一项
  4. 最后一个直接选择，没有时间算
  5. 笑脸公式+两期比重差，太耗时了，猜猜猜猜
