翻译推理

题型特征

  1. 题干和选项中出现比较明显的逻辑关联词
  2. 提问方式是“可以推出”“不能推出”等

解题思维

  1. 先翻译 == （逻辑关系词 翻译为 A => B )
  2. 再推理 == （根据逻辑公式进行推导）

前推后

关联词

  1. 如果。那么/就。
  2. 若。则。
  3. 只要。就。
  4. 所有。都。
  5. 为了。一定/必须。
  6. 。是。的充分条件

后推前

谁必不可少 就放在箭头后边

且 或

德 摩根定律

推理方式

提问方式：以下哪项中的推理形式/结构与题干中的推理形式/结构相同？

照这写就行

组合排列

题型特征

  1. 题干给出两组以及两组以上对象
  2. 给出几组对象之间的关系

排除法、代入法

  1. 何时用：题干条件为真，选项信息充分。
  2. 如何用：读一句，排一句

辅助技巧

日常结论

类似于言语的细节判断

题干： 言语、无逻辑关联词

提问：由此可以推出、不能推出

解题思路
  1. 不选
    1. 逻辑错误
    2. 无中生有
    3. 偷换概念
  2. 慎选
    1. 比较关系：比。, 越来越。、更
    2. 绝对词：一定、必须、肯定、只要就、只有才
    3. 程度：最、极大、很。
    4. 范围的扩大
  3. 优选
    1. 词汇：可能、有的、有些。

警惕五大坑王：更！最！越！首！极！

概念

消弱题型：反对

加强题型：赞同

  2. 论点与论据话题不一致

    1. 论点：文明。
    2. 论据：伐木。
    3. 搭桥：伐木 与 文明的关系

否定论据

❌易错点❌

对策

选项中出现对策类的词汇，并不知道对策是否执行，所以一定要注意，有问题–不选择
如：

  1. 法律 应该 制定 …
  2. 企业 应当 担负 …
诉诸无知 == 不知道

不知道的论据，不能当论据了，不选择。如：

  1. 没有实际数据显示。..
  2. 没有研究表明。…
  3. 我们没注意到。..
超出范围

如果讨论的是 [A, B], 结果来了个论据是 还有更好的，超出谈论范围，不选择

无关选项

和话题、主体不一致的不选择

类比的选项

类比一般比较弱，慎选 如：两者类似
