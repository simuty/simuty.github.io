    代入排除法

    适用代入排除法的典型题型

    1. 多位数： 几个数组成的
    2. 年龄： 选择各个年龄值
    3. 不定方程：下一部分
    4. 余数：有总数、有剩余、分东西问题

    方程与不定方程

    1. 找等量关系
      1. 已知总和
      2. 已知 A、B 之间的关系：A 比 B..,A 是 B…,A 与 B…
      3. 隐藏在多个条件中的不变量
    2. 根据等量关系设未知数，列方程，
    3. 解方程

    不定方程

    • 特征
    • 奇偶性
    • 尾数
    • 倍数
    • 实例
    • 非整数

    识别特征：未知数的个数多于方程的个数
    常见类型：两个未知数一个方程，三个未知数两个方程

    步骤

    1. 列不定方程; 再简单的问题也列出来，
    2. 数据小就试试；
    3. 分析数字特征；
    1. 如果求两个未知数，直接带入法
    2. 枚举法，数据小，就一个个试试

    概念

    1. 偶数：一个整数可以被 2 整除
    2. 奇数：一个整数不是偶数
    3. 质数：只能被 1 & 本身整除，2、3、5、7、11
    4. 合数：除了 1 & 本身以外，还能被其他整数整除 4\6\8\9

    2 是唯一一个偶数质数

    运算规律

    1. 奇偶相加减 = 奇数
    2. 奇数+奇数 = 奇数
    3. 偶数+偶数 = 偶数
    4. 相乘有偶数 = 偶数
    5. 相乘无偶数 = 奇数

    例： 3X + 2Y = 10

    1. 2Y = 偶数
    2. 3X 得是偶数
    3. X 只能是偶数

    尾数为 0、5，可以分析尾数法

    3X + 10Y = 61

    1. 10Y 的尾数是 0
    2. 3X 的尾数只能是 1
    3. X = 7 17
    4. Y = 4 1

    3X + 8Y = 44

    1. 没有共同的数的倍数是 3 44
    2. 4 的倍数 8 44
    3. 得出两项都是 4 的倍数，那么 3X 也是 4 的倍数
    4. X = 4 的倍数 [4 8 12]
