MacTool

Mac 开源免费工具汇总, 只罗列开源好用的。
更全列表请参考awesome-mac

必备

  • Homebrew - 体验通过命令行安装 Mac 软件的工具(大部分是命令行工具)。Open-Source Software Freeware
  • Homebrew Cask - 基于Homebrew扩展的，通过命令行安装 Mac GUI 软件的工具。Open-Source Software Freeware
  • Homebrew search software
  • 中科大换源
  • VMware Fusion–虚拟机个人用户免费

安全工具

  • LuLu - 免费的macOS防火墙，旨在阻止未经授权（传出）的网络流量。

浏览器

  • Firefox 国际版 建议FTP下载
  • Brave 用 Brave 浏览更快更安全
  • Chrome google chrome

输入法

  • 强烈建议用自带的

实用工具

系统工具

  • Amphetamine - 覆盖您的节能设置并让您的Mac保持清醒状态。
  • 🚩OmniDiskSweeper — 分析文件大小
  • SwitchHosts — 切换hosts

终端

  • 🚩Tmux—Tmux和Vim一样属于字符终端软件，不需要任何GUI的支持，在远程登录时尤其有用。
    • 📖博文： tmux使用说明
  • iTerm2 — iTerm是一个GUI软件,简单好用。

菜单栏工具

  • HiddenBar - 隐藏菜单栏图标。Freeware Open-Source Software

剪贴板工具

  • Paste - 智能剪贴板历史片段管理
  • CopyQ - 高级功能剪贴板管理工具

窗口工具

  • ShiftIt - 窗口位置和大小管理软件。
  • Spectacle - 简单的移动和调整大小的窗口，和可定制的键盘快捷键

截图工具

  • Snipaste - 一个简单但强大的截图工具。Freeware

下载工具

  • Motrix - Motrix 是一款全能的下载工具，支持下载 HTTP、FTP、BT、磁力链、百度网盘等资源。 Open-Source Software Freeware
  • qBittorrent - 一个替代 μTorrent 的开源软件。 Open-Source Software Freeware
  • Transmission - 免费的 BitTorrent 客户端 Open-Source Software Freeware
  • You-Get - 网络富媒体命令行下载工具。Open-Source Software Freeware

浏览器插件

整理了一份全面的–chrome使用指南

  • Saladict 沙拉查词
  • Octotree – github侧边栏
